# Навыки: 3D и Серверы

## 1. 3D Моделирование и Печать
Имею опыт работы инженером-программистом (1.5 года), включая обслуживание 3D-техники.

*   **Софт:** Autodesk Maya, Blender.
*   **Оборудование:** Работал с принтером Hercules H2 (FDM).
*   **Применение в медицине:** Печать брендированных кейсов для таблеток (таблетниц) или корпусов для датчиков.

## 2. Требования к серверам

### Малые проекты (Лендинги, MVP)
*   **Конфигурация:** VPS 2 vCPU / 4 GB RAM / SSD.
*   **Схема:** All-in-one (Веб-сервер + БД на одной машине).

### Крупные проекты (МИС, Нагруженные системы)
*   **Требование:** Отказоустойчивость.
*   **Схема:** Разнесение сервисов.
    *   Отдельный сервер БД (Master + Slave для репликации).
    *   Отдельные App-серверы.
    *   Балансировщик нагрузки.
    *   Отдельный сервер для бэкапов.
