# Техническое задание: СЭД (Документооборот)

## 1. Суть проекта
Система для создания, согласования и хранения электронных документов.

## 2. Функционал
*   **Маршруты:** Настройка цепочек согласования (Составитель -> Юрист -> Директор).
*   **ЭЦП:** Подписание документов электронной подписью.
*   **Поиск:** Поиск по атрибутам и тексту документа.

## 3. Мои задачи (Backend)
1.  **Workflow движок:** Реализация логики переходов документа по статусам.
2.  **Валидация:** Проверка подписей ЭЦП на бэкенде.
3.  **Архив:** Организация загрузки и хранения файлов.
