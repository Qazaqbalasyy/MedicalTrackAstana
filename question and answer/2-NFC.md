# Техническое задание: Платформа NFC-визиток

## 1. Суть проекта
Веб-сервис для создания цифровых профилей пользователей, привязанных к NFC-меткам.

## 2. Функционал
*   **Публичная страница:** Фото, контакты, кнопка "Сохранить в телефон" (vCard).
*   **Личный кабинет:** Редактирование данных, смена фото, скрытие полей.
*   **Админка:** Привязка ID метки к пользователю, блокировка.

## 3. Стек
*   PHP 8+, MySQL.
*   Frontend: HTML/CSS/JS (без фреймворков).

## 4. Мои задачи (Backend)
В данном проекте выполняю роль бэкенд-разработчика.
1.  **База данных:** Проектирование схемы (users, links, cards).
2.  **vCard:** Написание скрипта генерации .vcf файла для импорта контактов в телефон.
3.  **Логика:** Обработка QR-кодов и редиректов.
