# Техническое задание: МИС (Медицинская Система)

## 1. Суть проекта
Информационная система для автоматизации клиники: регистратура, прием врача, касса.

## 2. Основные модули
*   **Регистратура:** Сетка расписания врачей, запись пациентов.
*   **Электронная карта:** Истории болезней, шаблоны осмотров.
*   **Касса:** Учет услуг и оплат.

## 3. Стек
*   Backend: PHP (Laravel/Symfony).
*   Database: PostgreSQL.

## 4. Мои задачи (Backend)
Отвечаю за серверную логику и хранение данных.
1.  **Структура БД:** Нормализация данных пациентов и приемов.
2.  **API:** Создание методов для фронтенда (получение расписания, сохранение осмотра).
3.  **Логика:** Предотвращение накладок в расписании (блокировки транзакций).
4.  **Безопасность:** Разграничение прав доступа (Врач/Админ).
