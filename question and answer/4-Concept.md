# Концепция: Автоматизация клиники (IoT)

## 1. Идеи автоматизации
Внедрение датчиков для контроля параметров среды.

### Сценарии:
*   **Холодильники:** Контроль температуры хранения лекарств.
*   **Кабинеты:** Мониторинг CO2 (качество воздуха).
*   **Оборудование:** Учет перемещения дорогих аппаратов (BLE метки).

## 2. Техническая реализация
*   **Датчики:** Передача данных по Wi-Fi/Bluetooth.
*   **Протокол:** MQTT (легковесный протокол обмена сообщениями).
*   **Сервер:** Прием данных и запись в БД.

## 3. Мои задачи
1.  **Сбор данных:** Написание скрипта-worker'а, подписывающегося на топики MQTT.
2.  **Обработка:** Валидация входящих данных, фильтрация шума.
3.  **Алертинг:** Логика отправки уведомлений администратору при выходе значений за пределы нормы.
