# Параметры ПК разработчика

Текущая конфигурация ноутбука:

*   **Модель:** HP Pavilion Gaming (примерно)
*   **CPU:** Intel Core i5-9300H (4 ядра, 2.40 GHz)
*   **RAM:** 8 GB
*   **GPU:** NVIDIA GeForce GTX 1650
*   **ОС:** Windows + WSL 2 (Linux подсистема)

Этой мощности достаточно для:
1.  Запуска веб-сервера (Nginx/Apache) и PHP.
2.  Работы с Docker контейнерами.
3.  3D-моделирования средней сложности (Blender).
